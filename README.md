# wealthyleaf_UI

## Setup & Start

1. **Install dependencies:**
   ```sh
   npm install
   # or
   yarn install
   ```

2. **Configure Firebase:**
   - Add your Firebase configuration to `src/firebase/config.js`.

3. **Start the development server:**
   ```sh
   npm start
   # or
   yarn start
   ```

4. **Open the app in your browser:**
   ```
   http://localhost:3000
   ```
